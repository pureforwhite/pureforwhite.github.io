Java 物件導向編程：Inheritance繼承

I. 繼承: 定義

繼承是物件導向編程的基本原則之一。它允許一個類別（子類別）繼承另一個類別（父類別）的屬性和方法。繼承的主要優點是它促進了程式碼的可重用性。

II. Java 中的繼承

在 Java 中，我們使用 extends 關鍵字表示繼承。以下是一個簡單的範例：

public class Animal { public void eat() { System.out.println("This animal eats food."); } } public class Dog extends Animal { public void bark() { System.out.println("The dog barks."); } }

在此範例中，Dog 是子類別，Animal 是父類別。Dog 類別繼承了 Animal 類別的 eat 方法。因此，即使在 Dog 類別中未定義 eat，你也可以建立一個 Dog 物件並在其上呼叫 eat 方法。

III. super 關鍵字

Java 提供了 super 關鍵字，用於引用直接的父類別物件。例如，如果我們想從子類別中呼叫父類別的方法，我們可以使用 super.methodName()。

IV. 方法覆寫

方法覆寫允許子類別提供一個已由其父類別提供的方法的具體實現。子類別中的方法必須具有與父類別中的方法相同的名稱、返回類型和參數。

public class Animal { public void makeSound() { System.out.println("The animal makes a sound."); } } public class Dog extends Animal { @Override public void makeSound() { System.out.println("The dog barks."); } }

在此範例中，Dog 類別覆寫了來自 Animal 類別的 makeSound 方法。

這是 Java 中繼承的基本概述。實習創建你自己的類別並使用繼承，以更好地理解這個概念。
